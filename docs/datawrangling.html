3  Data Wrangling

The Data is available on the SBA.gov website. It’s in tabular format (csv) and divided into 14 files, 1 of which is a dictionary. totaling about 5GB of data. The SBA includes a file with all of the variable definitions which I have included in the first tab set below (with some additional context that was missing).

The dictionary along with some notes I’ve added can be seen here.

3.0.0.1 Preparing the Data

We’ll need to use several libraries to perform various cleaning and wrangling functions.

library(tidyverse)
library(ggplot2)
library(stringr)
library(purrr)
library(dplyr)
library(infer)
library(sparklyr)

Initially, I loaded the data onto RStudio. Each file contained 900,000 observations and 53 columns/variables. To make the process more manageable, I used reduce() from the purrr package, to condense the files into 4. Where each file would undergo the same basic cleaning process.

We’ll continue the wrangling using the sparklyr package for the sake of my aged computer, at least until the data is more refined and manageable.

businesstype_fix<- function(df){
  df %>%   
  mutate(BusinessType = case_when(
      BusinessType == "Non-Profit Childcare Center" ~ "501(k) Childcare Organization",
      BusinessType == "501(c) Non Profit except 3,4,6,19" ~ "501(c) Non Profit except 3,4,6,19",
      BusinessType == "501(c)19 Non Profit Veterans" ~ "501(c)19 Non Profit Veterans",
      BusinessType == "501(c)3 Non Profit" ~ "501(c)3 Non Profit",
      BusinessType == "501(c)6 Non Profit Membership" ~ "501(c)6 Non Profit Membership",
      BusinessType == "Non-Profit Organization" ~ "501(c)3 Non Profit",
      TRUE ~ BusinessType))}
